Para executar o sistema executar o seguinte comando na consola:

	make

O dados de configuração encontram-se no ficheiro "simulador.conf", todos os tempos estão em segundos e os restantes dados em unidades. A ordem dos dados é indiferente.

O ficheiro "config.h", contém todas as bibliotecas utilizadas e dados relativos aos sockets. Os ficheiros "monitor.c" e "simulador.c", implementam, respetivamente, o monitor e o simulador.

Os dados da simulação poderão ser encontrados em "monitor.log".
