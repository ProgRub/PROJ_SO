O utilizador deve executar o comando "make" no terminal na diretoria do projeto para serem criados os executáveis.
Para começar a simulação o utilizador deve abrir dois terminais.
Para não dar erro na socket é necessário executar primeiro o simulador com o comando "./Simulador" num dos terminais.
Depois no outro terminal executar o monitor com o comando "./Monitor".
Escolher a opção para iniciar a simulação no monitor e depois pressionar space + enter no simulador.
Depois pressionar as opções 3 e 4 para ver os eventos em tempo real e as estatísticas.
Quando terminar a simulação deve sair do monitor pressionando a opção 2.
